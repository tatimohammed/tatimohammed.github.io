Breast Cancer Classification

 This is a Machine Learning & Data Science Project that we will try to build a predicting model to classify the malignancy of a breast cancer tumor using ultrasound images converted to mathematical features. The model will use this related data of a particular tumor. Of course, we will have to clean the data & make a simple EDA to understand it more.
Tools used: Python -> (pandas - requests - Matplotlib - Seaborn - Sklearn - Numpy - Pickle - Json - Flask) - JQuery - JavaScript - HTML - CSS - Postman - VSCode IDE - Jupyter Notebook

Dice Game (JavaEE Servlets)

 In this repository you will find a Dice game developed using jakarta.servlets. The main goal is to get familier with Servlets, Sessions, Context ... We are managing a users and their login & logout also thier scores.
Tools used: Java -> (Java EE - Servlet - Tomcat Server) - Eclipse IDE

School Database Management

 It is a school project that aims to create a SQL script that inserts data given in CSV format into a database along with the auto-generating of the PRIMARY KEY. In the other part of the project, based on the SQL script that we got, we need to import the data into a NoSQL database, I've choosen MongoDB to do so.
Tools used: Python -> (pandas - datetime) - SQL - Studio 3T - MongoDB - MySQL - VSCode
